# Recommended download
Below is the recommended ways to download/clone the repo to speed up or reduce the bandwidth usage

```
1) git clone https://github.com/FiuuPayment/Mobile-XDK-Fiuu_Examples.git --depth 1

2) svn export https://github.com/FiuuPayment/Mobile-XDK-Fiuu_Examples.git/trunk/(TARGETED_FOLDER)

3) curl https://codeload.github.com/FiuuPayment/Mobile-XDK-Fiuu_Examples/tar.gz/master | tar -xz --strip=2 Mobile-XDK-Fiuu_Examples-master/(TARGETED_FOLDER)
```
